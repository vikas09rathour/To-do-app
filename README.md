# Todo App

A simple and responsive Todo application built using React and Vite. It helps users manage daily tasks efficiently.

## Features

* Add tasks
* Delete tasks
* Mark tasks as completed

## Tech Stack

* React
* Vite
* JavaScript
* CSS

## Installation

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
npm install
npm run dev
```
